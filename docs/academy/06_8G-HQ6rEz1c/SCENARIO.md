# Реверс-инжиниринг — Видео 06: «Apps and Integrations» (2:10)

Тема: каталог приложений/интеграций. Для ТЗ — §16 Settings → Apps & Integrations.

## Механика (из звука)
- **Apps page** в workspace settings — полный список поддерживаемых интеграций; browse по категориям + поиск по имени; список регулярно обновляется.
- Карточка приложения: overview возможностей, ссылки на ресурсы, разработчик.
- **Три типа приложений:** партнёрские (built by integration partners, напр. Granola), first-party (built by Attio, напр. PandaDocs), и от developer community.
- **Установка зависит от автора:**
  - Партнёрское (Granola): Install → завершение установки/конфигурации НА стороне партнёра.
  - First-party (PandaDocs): установка прямо внутри Attio → Install → connect account → finish setup.
- **Виджеты в record-страницах:** некоторые приложения дают виджет на record-страницу (настройка с app page или с record-страницы); на deal-записи виджет рядом с action-кнопками.
- **SDK** для своих приложений: docs.attio.com/sdk.

## Требования для ТЗ
- Страница **Apps** в settings: каталог (категории + поиск), карточка приложения (overview/resources/developer), Install.
- Типы интеграций (демо-каталог): Email/Calendar (Gmail/Outlook), Slack, Typeform, Mailchimp, Intercom, Segment, Granola, PandaDocs, Webhooks.
- Виджеты приложений на record-страницах.

## Сценарии (acceptance)
1. Settings → Apps → поиск/категория → карточка → Install → connect → виджет доступен на record.

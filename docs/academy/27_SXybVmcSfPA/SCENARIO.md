# Реверс-инжиниринг — Видео 27: «Notes, tasks, and email sending» (3:52)

Тема: продуктивность — comments, tasks, email, notes. Для ТЗ — §8 Record page (табы) + email-инструмент.

## Механика (из звука)
- **Comments** — на любой record/list entry; короткая заметка + опц. @tag коллеги → уведомление в Attio (+ email по настройкам); уведомления о реплаях.
- **Tasks** — to-do себе/коллеге: описание + assignee + due date; привязка к любому объекту; видны в task-табе записи и на общей **Task page** (фильтр по assignee); mark complete.
- **Email-инструмент** (при подключённом ящике): отправка письма с record-страницы; выбор получателей (main point of contact / все associated = team); с нуля или из **library of email templates** (напр. onboarding с booking-ссылкой); toggle mass send off → весь team в один тред; отправка без переключения вкладок.
  - **Mass send** — кастомизированные письма группе за раз (пример: всем кандидатам в стадии screening → interview-приглашение); переменные-атрибуты (first name) кастомизируют текст каждому.
  - **Email templates:** email tab → Templates; вставка переменной любого атрибута person или связанных объектов (company name).
- **Notes** — общее место для prep/контекста; на любой записи (чаще company); опц. линк к встрече календаря; с нуля или из workspace-шаблона; шаблоны создаются в notes tab → Templates.

## Требования для ТЗ
- Comments + @mention + уведомления (web/email).
- Tasks (assignee/due/linked record) + task-таб + Task page + complete.
- Email-инструмент: send с записи, выбор получателей (main contact/team), templates с переменными, mass send.
- Notes: на записи, линк к встрече, шаблоны.

## Сценарии (acceptance)
1. Comment с @mention → коллега получает уведомление.
2. Task assignee+due → на task-табе записи и Task page; complete.
3. Email с company-записи → выбрать team → шаблон onboarding → mass send off → один тред.
4. Mass send всем в стадии screening → template с {first name}.
